sudo -l ：查看当前用户可使用的命令，仅限root用户可用。

修改sudo配置文件/etc/sudoers
sudo为系统管理员提供配置文件，允许系统管理员集中地管理用户的使用权限和使用的主机，它默认的存放位置是/etc/sudoers。

查看并修改sudoers的文件权限
ls -l sudoers
[root@VM_16_3_centos etc]# ls -l sudoers
-rw-r----- 1 root root 3963 Feb 3 12:50 sudoers

发现sudoers是一个只读文件，若要修改，必须修改权限。
[root@VM_16_3_centos etc]# chmod 777 sudoers
[root@VM_16_3_centos etc]# ls -l sudoers
-rwxrwxrwx 1 root root 3963 Feb 3 12:50 sudoers
